相关函数说明
fnmatch函数：判断字符串是否与通配符匹配，fnmatch函数可判断传入该函数内的字符串是否与该函数内的通配符相匹配。
array fnmatch(string pattern, string string [, int flags])

参数pattern为必选参数，用于指定通配符。
参数string为必选参数，用于指定传入该函数内的字符串。
参数flags为可选参数，可以是FILE_USE_INCLUDE_PATH、FILE_APEND或LOCK_EX（获得一个独占锁定）。

示例：(用fnmatch()函数判断字符串“mingrisoft”是否与通配符“mingri*”相匹配)
