[极客大挑战 2019]Secret File

一个诡异的页面，没有任何按钮和链接，让人找他的秘密。。。
日常F12检查元素，发现一个链接

点进去，成功到达下一步

这个页面就有可点击的链接了

点进去居然就查阅结束，有点东西。返回上一个页面，重新点击，发现还是这样子。
所以F12，发现链接和跳转的网页不太一样


所以应该是被重定向跳转了。网上搜了一堆禁止重定向的方法，都不起作用，然而bp又用不熟，所以就卡在这一小会儿。后来我突然想到python爬虫有获取源码的方法，并且可以禁止重定向，所以试了一下，发现还真可以。

给了一个secr3t.php，进入查看。

提示说flag就在flag.php里。而且对传入的参数进行了过滤，一些伪协议变得不好使了。
先进入flag.php看看。

还是不行，提示说已经找到了，但是却看不到，那估计就是将flag藏在了php源码中，又以为php://filter协议没有被过滤，所以就可以php://filter/read=convert.base64-encode/resource=flag.php读取源码，这边必须要有base64编码，否则读取的源码会被当成php执行，依旧是看不到flag的。
最后将得到的一串东西解码即可得到flag
