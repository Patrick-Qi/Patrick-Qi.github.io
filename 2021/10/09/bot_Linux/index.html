前言

对于gocq和nonebot的详细介绍参考上篇文章Windows10系统下基于gocq和nonebot2框架实现的qq机器人
本篇主要基于Linux CentOS系统的服务器进行机器人的搭建

服务器的选择

市场上有很多服务商，阿里云腾讯云华为云等等，随便选一个都行。购买好之后可以通过xshell进行连接。一般Linux服务器是自带python的，可以在命令行输入python查询版本。

gocq配置

  1. 下载对应的go-cqhttp release文件
    下载对应版本压缩包.zip 或 .tar.gz
  2. 解压并运行 到这里便完成了gocq的下载与第一次启动。
  3. 配置config.yml文件
    此处修改同样适用config.hjson配置文件
  4. 启动gocq，第一次登录会要求扫码。
    到这里gocq配置完毕

###nonebot配置

  • pip install -i https://pypi.doubanio.com/simple nb-cli(推荐，方便后期生成Bot项目)
  • pip install -i https://pypi.doubanio.com/simple nonebot2
